# InsightLab — premium restyle (Nocturne dark + Daylight light), drop-in `src/`

Премиальный «soft-SaaS» вид (по референсу): крупные мягкие карточки с большими
радиусами и нежными тенями, много воздуха, чистая типографика, спокойный
серо-голубой фон в светлой теме и глубокий графит в тёмной. Фирменный синий
`#2D9CDB`. Обе темы, мгновенный переключатель, анимированный фон.

**Логика не тронута** — имена функций, пропсы, ключи состояния, вызовы Supabase
и система прав без изменений. Менялись только разметка, стили, анимации.

> ⚠️ Сначала бэкап рабочей версии: `git checkout -b nocturne-restyle`

---

## Установка

```bash
npm i framer-motion          # единственная новая зависимость (если ещё не стоит)
```
**Новых пакетов для этого этапа не требуется.** Логотип встроен как векторный
компонент (без загрузки картинок).

1. Скопируйте содержимое `delivery/` поверх проекта (структура повторяет ваш `src/`).
2. **Сохраните свой `src/lib/api.js`** — он ваш и не меняется.
3. `npm run dev`.

### Структура
```
delivery/
├─ index.html · package.json · vite.config.js
└─ src/
   ├─ main.jsx · nocturne.css           ← токены обеих тем + база
   ├─ Login.jsx · InsightLabCRM.jsx     ← перекрашены/переоформлены
   └─ components/
      ├─ theme.jsx                       ← ThemeProvider / useTheme / ThemeToggle
      ├─ NocturneBackground.jsx          ← анимированный фон

      └─ assets/                         ← png/svg логотипа (запас, приложением не нужны)
```

## Что изменилось визуально

- **Логотип.** Настоящий вектор `insightlab` встроен в `Logo.jsx` через
  `currentColor` — **сам перекрашивается под тему** (чёрный на светлой, белый на
  тёмной). Стоит в шапке и на экране входа. Растровые `assets/` — просто запас.
- **Дизайн-система → soft-SaaS.** Централизованно, через токены `--c-*` и общие
  примитивы (`Panel`, `Btn`, `Modal`, инпуты, `StatCard`, `KanbanCard`, колонки),
  поэтому новый стиль раскатался по **всем** разделам сразу:
  - карточки: радиус 20px, тонкая граница + лёгкий «sheen» сверху, мягкая
    многослойная тень, плавный подъём на hover;
  - кнопки/инпуты: радиус 12px, синяя кнопка со свечением, фокус-кольцо у полей;
  - канбан: воздушные колонки (радиус 18) с цветным маркером этапа и счётчиком-
    пилюлей, крупные карточки с акцентной кромкой и «приземлением» на hover;
  - аналитика: KPI-плитки крупнее (число 30px, табличные цифры);
  - шапка: «стеклянная» (blur), nav-пилюли, логотип, переключатель темы, аватар.
- **Светлая тема** освежена: спокойный серо-голубой градиент-фон вместо чистого
  белого, белые «парящие» карточки, мягкие сине-серые тени.
- **Тёмная тема** в том же духе: карточки чуть светлее (`#151D2D`), стекло, тонкое
  свечение; анимированный фон сохранён и приглушён.

## Производительность и доступность
- Переход между темами — лёгкий кросс-фейд только по дешёвым свойствам
  (цвет/фон/граница); тени/обводки в глобальный transition не входят, чтобы
  переключение не «дёргало» сотни узлов.
- Фон: rAF, ограниченный DPR, пауза на скрытой вкладке, `prefers-reduced-motion`.
- Контраст текста сохранён в обеих темах.

## Архитектура (как и раньше)
`C.* → var(--c-*)`; темы заданы в `nocturne.css` через `[data-theme="dark"|"light"]`
на `<html>`. Смена атрибута мгновенно перекрашивает весь интерфейс. Выбор темы
запоминается (профиль → localStorage → системная тема). Подробности — в
`components/theme.jsx`.
