# InsightLab — premium restyle (Lumen light + Nocturne dark), drop-in `src/`

## ✦ Обновление «Lumen» (светлая тема по умолчанию)

Системный рестайлинг через токены — раскатывается на все экраны автоматически,
логика не тронута:
- **Светлая тема по умолчанию** (лавандовый градиент `#EEF0F6→#E2E5F0`),
  переключатель тёмной сохранён.
- **Чёрные pill-кнопки** для главных действий (`Btn variant="primary"` →
  чёрный на светлой / почти-белый на тёмной), вторичные — белые pill
  с тонкой границей. Скругление кнопок — 999 (пилюля), ховер — подъём.
- **Индиго-акцент** `#5B5BFF` (токен `--c-indigo`); фирменный синий `#2D9CDB`
  остаётся на акцентах/графиках.
- **Настоящее прозрачное стекло** на карточках/панелях + цветные blobs фона
  (см. раздел ниже). Поля ввода — скругление 14, фокус-свечение.

Главное: `PAGE`-обёртка сделана прозрачной, чтобы лавандовый фон и blobs
с `body` просвечивали сквозь стекло карточек.

### Экран «Продажи» — под концепт Lumen

Визуал собран поверх существующей логики (поиск/фильтр/KPI/DnD/импорт — не тронуты):
- **Шапка**: логотип-wordmark (без плашки «CRM»), нав-пилюли (активная — лавандовая),
  строка поиска «Поиск по лидам…» прямо в шапке (на странице продаж), переключатель
  темы, колокольчик с точкой, круглый аватар с инициалами. Аватар — это и есть
  переключатель роли (прозрачный `select` поверх), вся демо-логика сохранена.
- **Заголовок**: «Продажи» 32/800 + подзаголовок «Воронка лидов · N активных сделок на X».
- **Панель действий**: сегментированный тогл «Активные / Все» (тёмная пилюля, локальный
  фильтр), кнопки CSV / XLSX / Импорт, чёрная pill «+ Лид».
- **KPI (4 стеклянные карточки)**: Win-rate (зелёный), Пайплайн (тёмный), Выручка
  (индиго), Средний чек (последняя — с лёгким голубым градиентом). Считаются из ваших лидов.
- **Колонки канбана** — без «коробки»: цветная точка + название + счётчик; карточки лежат
  прямо на фоне; внизу каждой колонки стеклянная кнопка «+ Добавить» (создаёт лид в этой
  стадии). Подсветка зоны при drag.
- **Карточки лида**: название жирным, кнопка «···» в углу (видна при наведении), тег-пилюля
  источника (тихая), «🔥 Горячий» оранжевым (когда касание сегодня/просрочено), сумма
  справа, футер с иконкой-часами и статусом, левая цветная акцент-полоса по стадии.

Превью реального экрана (обе темы, ☀/🌙): **`InsightLab - Sales Screen Preview.html`**.

### Исправление «стекло не читается» (живой цветной фон)

На проде фон был почти однотонным, поэтому стеклу было не на чём «играть».
Причина: `NocturneBackground` (canvas) подключён только на экране интервью/
входа, а на основных экранах цвет шёл только из слабых `--c-blobs` на `body`.

Что сделано (только CSS, логика не тронута):
- **Фиксированный цветной mesh-слой** `body::before` (position:fixed, под контентом,
  с `blur(28px)`) — именно его сэмплит каждый `backdrop-filter` выше, так что
  цвет реально виден сквозь карточки на ВСЕХ экранах.
- **Яркие blobs** (`--c-blobs`): тёмная — индиго/синий/фиолет/бирюза (alpha 0.34–0.55),
  светлая — нежные пастельные (0.40–0.50). Было 0.10–0.22 — почти невидно.
- **Иерархия стекла**: колонка (`--g-col`, прозрачнее, blur 14) → карточка
  (`--g-card`, плотнее, blur 22–24, яркий кант + верхний блик). Текст остаётся чётким.

> разные иконки статуса, стеклянная плашка «+ Добавить») можно добавить
> прямо в разметку карточки лида/респондента — скажите, сделаю

## Установка

```bash
npm i framer-motion          # единственная новая зависимость (если ещё не стоит)
```
**Новых пакетов для этого этапа не требуется.** Логотип встроен как векторный
компонент (без загрузки картинок).

1. Скопируйте содержимое `delivery/` поверх проекта (структура повторяет ваш `src/`).
2. **Сохраните свой `src/lib/api.js`** — он ваш и не меняется.
3. `npm run dev`.

## Что изменилось визуально

- **Логотип.** Настоящий вектор `insightlab` встроен в `Logo.jsx` через
  `currentColor` — **сам перекрашивается под тему** (чёрный на светлой, белый на
  тёмной). Стоит в шапке и на экране входа. Растровые `assets/` — просто запас.
- **Дизайн-система → soft-SaaS.** Централизованно, через токены `--c-*` и общие
  примитивы (`Panel`, `Btn`, `Modal`, инпуты, `StatCard`, `KanbanCard`, колонки),
  поэтому новый стиль раскатался по **всем** разделам сразу:
  - карточки: радиус 20px, тонкая граница + лёгкий «sheen» сверху, мягкая
    многослойная тень, плавный подъём на hover;
  - кнопки/инпуты: радиус 12px, синяя кнопка со свечением, фокус-кольцо у полей;
  - канбан: воздушные колонки (радиус 18) с цветным маркером этапа и счётчиком-
    пилюлей, крупные карточки с акцентной кромкой и «приземлением» на hover;
  - аналитика: KPI-плитки крупнее (число 30px, табличные цифры);
  - шапка: «стеклянная» (blur), nav-пилюли, логотип, переключатель темы, аватар.
- **Светлая тема** освежена: спокойный серо-голубой градиент-фон вместо чистого
  белого, белые «парящие» карточки, мягкие сине-серые тени.
- **Тёмная тема** в том же духе: карточки чуть светлее (`#151D2D`), стекло, тонкое
  свечение; анимированный фон сохранён и приглушён.

## ✦ Усиленный glassmorphism (карточки = настоящее стекло)

Карточки и крупные панели теперь **по-настоящему прозрачное стекло** — сквозь них
просвечивает цветной фон страницы, как за стеклянной картой.

- **Цветной «жидкий» фон страницы.** В `body` добавлены мягкие размытые цветные
  пятна (blobs) индиго `#5B5BFF` / голубой `#2D9CDB` / сиреневый `#A78BFA`,
  приглушённые — токен `--c-blobs`. Их видно сквозь стекло, они дают эффект
  преломления. На светлой теме мягче, на тёмной — атмосферно-тусклые.
- **Стеклянные карточки** — класс `.glass-card` + токены `--g-*`:
  - фон полупрозрачный (`--g-card`: светлая `rgba(255,255,255,0.16)`, тёмная `0.06`);
  - сильное размытие за стеклом `backdrop-filter: blur(28px) saturate(140%)`;
  - светлый «кант» по краю `1px solid rgba(255,255,255,0.6)` + верхний внутренний
    хайлайт `inset 0 1px 0 rgba(255,255,255,0.5)`;
  - мягкая тень отрыва `0 8px 32px rgba(30,30,60,0.12)`, на hover — подъём `-2px`
    и усиление тени/блеска.
  - применено через примитивы **`Panel`** (аналитика, календарь, пользователи,
    настройки, дашборды) и **`KanbanCard`** (карточки лидов/респондентов) —
    раскатилось по всем экранам автоматически.
- **Колонки канбана** сделаны лёгкими полупрозрачными (`--g-col`), чтобы blobs
  просвечивали сквозь карточки; подсветка колонки при drag — `--g-col-over`.
- **Читаемость сохранена:** текст остаётся near-black `#0A0A0B` / светлым на тёмной;
  blobs приглушены; **модалки и поля ввода остаются плотными** (не стекло), чтобы
  формы и таблицы читались чётко. Утилита `.glass-strong` — для более плотной
  стеклянной подложки под тяжёлыми блоками текста, если понадобится.
- **Fallback:** где `backdrop-filter` не поддерживается — карточки автоматически
  становятся плотными (`@supports`), данные не теряются.

Посмотреть эффект вживую: **`InsightLab - Glass Cards Preview.html`** (обе темы,
кнопки ☀/🌙 справа).

## Производительность и доступность
- Переход между темами — лёгкий кросс-фейд только по дешёвым свойствам
  (цвет/фон/граница); тени/обводки в глобальный transition не входят, чтобы
  переключение не «дёргало» сотни узлов.
- Фон: rAF, ограниченный DPR, пауза на скрытой вкладке, `prefers-reduced-motion`.
- Контраст текста сохранён в обеих темах.

## Архитектура (как и раньше)
`C.* → var(--c-*)`; темы заданы в `nocturne.css` через `[data-theme="dark"|"light"]`
на `<html>`. Смена атрибута мгновенно перекрашивает весь интерфейс. Выбор темы
запоминается (профиль → localStorage → системная тема). Подробности — в
`components/theme.jsx`.
